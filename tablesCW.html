Bordered Table With Centered Content

The w3-border class defines a bordered table, this tag has to be used after w3-table, w3-center class will center table content:

First Name Last Name Points
Jill Smith 50
Eve Jackson 94
Adam Johnson 67

Striped Table with Borders Around Cells

The w3-striped class defines a striped table, w3-bordered class places borders around each cell:

First Name Last Name Points
Jill Smith 50
Eve Jackson 94
Adam Johnson 67

Table With All: borders around, bordered cells and striped.

The w3-table-all class sets all properties: borders and stripes.

First Name Last Name Points
Jill Smith 50
Eve Jackson 94
Adam Johnson 67

Hoverable Table with Colored Header

The w3-hoverable class sets allows for color change on hover. To change color of the header or any other element, just add class w3-color to that element.
